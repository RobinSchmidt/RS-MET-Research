\medskip
This is a verbatim environment:
\begin{verbatim}
int sum = 0;
for(int i = 1; i < 10; i++)
  sum += i;
print(sum);
\end{verbatim}


This is a tabular environment with two left-justified columns:

\begin{tabular}{ ll }
Left column
&
Right column
\end{tabular}

A blank line before and after the tabular is needed. Without it, the environment would appear in the text.


\medskip
This is a tabular environment with two columns, each holding a verbatim environment:

\begin{tabular}{ m{7cm}  m{7cm} }
\begin{verbatim}
int sum = 0;
for(int i = 1; i < 10; i++)
  sum += i;
print(sum);
\end{verbatim}
& 
\begin{verbatim}
int prod = 1;
for(int 1 = 0; i < 10; i++)
  prod *= i;
print(prod);
\end{verbatim}
\end{tabular}


These are quite finicky to create if you don't know how. One needs to specify column widths rather than alignments.
